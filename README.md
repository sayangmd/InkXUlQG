# 项目简介

基于深度学习的YOLOv12晶圆体缺陷识别检测系统，运用先进的目标检测技术，实现对晶圆表面缺陷的实时监测与分类。系统包含YOLO数据集、用户友好的UI界面以及登录注册功能，技术特点在于采用高效的YOLOv12算法，提高检测速度与准确性。此系统不涉及售卖交易，注重于技术实现与应用，为晶圆制造领域提供智能化解决方案。项目源码开放，以便技术交流与深入研究。


## 核心功能
- 基于YOLOv12深度学习算法的晶圆体缺陷实时监测与分类
- 提供YOLO数据集用于模型训练与优化
- 集成用户友好的UI界面，便于操作与交互
- 包含完整的登录注册功能，保障系统访问安全
- 开放项目源码，支持技术交流与深入研究

## 适用场景
- 晶圆制造过程中的质量控制与缺陷检测
- 半导体生产线的自动化监控
- 工业质检系统的智能化升级
- 深度学习目标检测技术的研究与应用

## 技术特点
- 采用先进的YOLOv12算法，提升检测速度与准确性
- 支持实时处理与分类多种晶圆表面缺陷类型
- 模块化设计，便于功能扩展与系统维护
- 开源架构，促进技术共享与社区协作
